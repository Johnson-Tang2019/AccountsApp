# 自动记账 Android App

一个本地优先的安卓记账原型。用户手动开启无障碍服务后，应用只会在支付宝或微信的支付成功页面解析金额、商户和时间，写入本机 SQLite 数据库。

## 构建

使用 Android Studio 打开项目，安装 Android SDK 35 后运行 `app`。首次启动点击“开启无障碍自动记账”，在系统设置中找到“支付信息自动记账”并开启。

## 隐私边界

- 仅处理 `com.eg.android.AlipayGphone` 与 `com.tencent.mm`。
- 跳过密码节点及所有可编辑输入框。
- 不申请网络权限，数据不上传；Android 备份也已关闭。
- 支付页面结构变化时，需相应调整解析规则。

> 无障碍能力应仅用于用户明确知情并主动授权的用途。发布到应用商店前，请核对目标市场的无障碍 API 政策、隐私披露和数据安全表单要求。
